设计原则
====================
大原则：认定原版是合理的，不可过于否定原版内容
  - 本 Mod 的定位就是原版补充而不会替代原版
  - 原版内容的定位与以前不变，降低上手难度

细则：
1. 任何方块不可从材料消耗、性能上全方位优于原版
2. 保持原版单位的定位，不要在前中期就出现随便打 T4 T5 的防御
3. 物品、液体要尽量少，降低复杂度
4. 慎重设计‘原版增强’类的工厂，此类工厂若设计不当，可能会破坏原版材料稀有度，降低游戏性

内容大纲
-------------------
以‘高维世界战争碎片’为噱头引出各种科技线，反正时间空间科技都出来了，未来科技、远古科技、其他世界线的科技也就可以有了呗。
预计有几大科技线：
- 次元科技线
  - 以时间、空间、离子科技为主题
  - 以后期为主，贵，慢，强

- 传统科技线
  - 以现实中的材料为主题（钢铁、钻石、铀等）
  - 以中期为主，包含一些比原版稍微强一点点的建筑

- 孢子科技线
  - 以生物为主题的科技（没错孢子可以随便转化成哺乳类细胞、菌类细胞、植物细胞）
  - 进攻强大，防守一般，比别的科技线有更多的单位容量

- 魔法科技线
  - 魔法哦哦
  - 华而不实，随机性大

- 混合 / 通用科技
  - 不属于几大科技线，或者同时使用几大科技线的材料组成的科技
  - 核心建造机、粒子对撞机（产生一些不可产生的材料，比如缺少矿物的地图）、废料回收机
  - T6+ 单位（每个科技线预计都有 T6 单位，混合的允许更高一层）
  - 对应 T6+ 的炮塔

从次元科技开始设计，最后搞魔法科技；
科技线全部差不多之后，再设计 T6 级科技。

次元碎片
====================
也许是更高维度发生了争斗或什么原因，我们发现了一些次元碎片，这些碎片具有超越我们所处维度的力量，似乎可以操作时间和空间...

## 物品
- 次元碎片
    - 爆钻采集
    - 设定上比较硬，有玻璃特性
    - 作为弹药击退很强

- 空间结晶
    - 次元碎片 + 硅 + 钍 + 矿渣
    - 可控的压缩与膨胀

- 时间结晶
    - 次元碎片 + 塑钢 + 相织 + 冷却液
    - 可控的物质活性

- 硬钍合金
    - 空间结晶 + 钍 + 石墨，用于围墙和弹药
    - 硬的要命，可能是被不可逆压缩了

- 时空合金
    - 空间结晶 + 时间结晶 + 巨浪合金 + 离子载液
    - 能炸
    - 能放电
    - 具有空间和时间结晶的特性

## 液体
- 时间流（时间结晶 + 冷冻液，用于冷却和超速）
- 离子载液（时间流 + 巨浪合金，用于武器）

## 炮台
### 原版更改
- 为分裂炮、气旋、浪涌，增加空间碎片弹药，接近塑钢的伤害，但击退很高
- 为连射、幽灵增加硬钍合金弹药，伤害约等于 钍 * 1.5
- 为噩兆增加空间结晶弹药，具有黑洞效果
- 为噩兆增加时空合金弹药，具有光棱塔效果（太屌取消）
- 为气旋增加时空合金弹药，类似更屌的巨浪（太屌取消）
- 滋水枪增加两种液体，离子载液伤害高一点，达到合金气旋差不多的水平

### 新炮
- 电子团炮（3级，发射‘小型电子团’）
- 离子炮（4级，类似 TAU 子弹，爆炸范围较大）
- 重型离子激光炮（5级，使用离子载液，末端爆炸的激光，中间出现少量小型电子团）
- 重型电子团炮（5级，射击时空合金，发射大型电子团）

## 钻头
- 硬钍钻头，加大量水超级快

## 运输

- 超空间提取器（提取器 + 相织 + 时空合金）

- 5x5: 三连发小飞机，可以初级过载，增强攻击速度、移动速度、挖矿速度、建造速度
- 6x6: 三连发带电小飞机，技能：
    - 高级过载：攻击速度、移动速度、挖矿速度、建造速度 都多加点
    - 瞬闪：瞬移，对沿途造成少量伤害
    - 逃跑：瞬移到3秒前自己的位置

### 陆地辅助线
- T4 提供血量大约等于绿色t3 的反射护盾，使用直流炮，通过技能提供加速状态

### 陆地进攻线（爬虫也先算进攻）
- T4 使用中型离子炮，死亡爆炸
- T5 使用重型离子炮，死亡爆炸，离子激光武器技能

### 空中辅助线
- T4 高速建造机 + 环绕离子团
- T5 反射护盾 + 跳跃

### 空中进攻线
- T4 轻型离子炮 + flare 攻击模式 + 摔下来爆炸
- T5 若干炸弹传送器，单体反弹护盾技能，优先拆塔

坚实工业 （科技线2）
====================
高维战争引起的时空错乱，使得 Sepluo 周边行星受到较大影响，一些已成为历史的资源及相应科技重新出现在我们的星系。
令人高兴的是，这些资源和技术与我们当前的技术非常类似，非常容易利用，可以极大地丰富我们的武器装备。

## 设计：
- 以中期和特色单位为主
- 最难制造的材料不会超过巨浪合金

## 核心元素：
- 铁（类似铜）
- 铝（类似铅）
- 铀（类似钍）

孢子入侵 （科技线3）
====================
被严重孢子化的 Ormos (名称摘自 Creeper World 3 第三章) 星球上，
受原始生物、次元碎片强辐射与外来生物质的影响，孢子逐渐分化、异变成各式各样的生物与非生物；
它们甚至侵蚀了 Ormos 上的核心，获得了相当程度的智能，不仅掌握了我们的科技，还结合我们的科技发展了更多孢子科技！
如果我们可以掌握孢子科技，一定能为我们带来更大的优势……

## 核心元素：
- 酸性组织液
- 有机液体

- 有机合金
- 霰石
- 苔藓块
- 腐化铜

## 建筑：
- 腐化铜钻
-
